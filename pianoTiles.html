Piano Tiles Prototype

A prototype for piano-tiles game but with a golf ball, where you have to control the ball to play the correct tune.

This project was made with Unity. It was inspired by the piano-tiles game on android. I added a bit of twist of mixing infinite runner with piano-tiles mechanism.

Basic Idea

This was also the early game/prototype I made while playing with Unity. I wanted to create an infinite runner like temple run but thought of adding music matchin mechanism like piano tiles to add a bit more challenge and make it more satisfying if the player played the correct tune.

From a technical standpoint, there is nothing fancy happening. The music tiles were 'quad' objects with a collision trigger. Hitting each tile played a note from the musical scale.

Hitting the golf ball consequtively with the tiles led a melody. The goal was to control the infinitely rolling ball and play the correct melody.

Retrospective

Although it was a prototype and not a complete game, this idea can be expanded upon with better visuals and different tile and terrain types to make it a proper game.
