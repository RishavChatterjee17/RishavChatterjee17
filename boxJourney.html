Puzzle Platformer

A orthographic camera puzzle platformer game wiht Unity.

This project was probably my first Unity project and it is very simple. The idea of the game is to avoid traps (red structures) while collecting nectar (yellow cylinders).

I was mainly experimenting with Unity animation system and collision system.

Basic Idea

Interestingly I wanted to add a story to the game. The player would collect nectars while avoiding traps and bring back a tree of life.

Tehcnically it was very simple to implement. The core gameplay was just avoiding obstacles which meant I could just replciate the last scene and create variations of the traps.

Since this was my first Unity project I learnt a lot about collisions, rigidbodys and how to add behavior to objects with scripts.

Retrospective

Although the idea and the gameplay was simple, this project helped me understand a lot about game-dev. With procedural level generation we could make the gameplay more fun and unique which I intend to implement in the near future.

I also thought of adding enemies that patrol the map to make the game more challenging.
